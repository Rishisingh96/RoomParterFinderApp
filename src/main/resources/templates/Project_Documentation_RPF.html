Problem Statement

  • Difficulty in finding compatible roommates
  • Lack of personalized filters (food, religion, occupation)
  • No trusted platform for verified room listings
  • Concerns about privacy and fake profiles

Objective

  • Match users with compatible room partners
  • Discover verified rooms/PGs with full details
  • Enable private and secure communication
  • Ensure verification and authenticity of all users

Core Features

1. Smart Room Partner Filter

Filter users by State, City, Area, Age, Gender, Food Type, Occupation, College, Religion

2. Verified Room/Hostel Suggestions

Display nearby rooms with Rent, Images, Address, Facilities, and Bookmarking

3. Communication Tools

WhatsApp, Email, Phone (with user consent), Chatbot (future)

4. Notifications

Email & SMS alerts for new matches and updates

5. Security & Privacy

OAuth login, profile verification, admin moderation

Technology Stack

  • Frontend: HTML, Tailwind CSS, JavaScript
  • Tools: Cloudinary (images), Google OAuth, GitHub Login
  • Backend: Spring Boot, Spring Security (OAuth2)
  • Database: MySQL
  • Auth: Google & GitHub OAuth, JWT (future)
  • Notification: SMTP (Email), Twilio (SMS), WhatsApp Cloud API
  • Chatbot: Dialogflow / AI-based (future)

Future Scope

  • Partner rating & review system
  • AI-based compatibility matching
  • Payment gateway integration
  • Admin dashboard for property management
  • Mobile App (Android/iOS)
